## 🎯 Mục tiêu khóa học

1. **Đọc lý thuyết** - Hiểu khái niệm trước khi code
2. **Thực hành ngay** - Code theo ví dụ trong bài
3. **Làm bài tập** - Củng cố kiến thức qua bài tập
4. **Hỏi khi thắc mắc** - Đừng ngần ngại hỏi thầy/cô
